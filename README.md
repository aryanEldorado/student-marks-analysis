# 📊 Student Marks Analysis

A beginner Python data analysis project using Pandas, Matplotlib, and Seaborn.

## 📌 What it does
- Calculates total marks, percentage, and grades
- Finds the topper and subject-wise toppers
- Visualizes data with bar charts and a heatmap
- Exports final data to CSV

## 🛠️ Libraries Used
- Python, Pandas, Matplotlib, Seaborn
